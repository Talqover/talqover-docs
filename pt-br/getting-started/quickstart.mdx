# Guia de Início Rápido

Comece a usar a plataforma Talkover AI Voice Agent em apenas 5 minutos. Este guia irá orientá-lo através da configuração do seu primeiro agente de voz e da realização da sua primeira chamada.

## Pré-requisitos

- Uma conta Talkover (cadastre-se em [app.talkover.ai](https://app.talkover.ai))
- Um número de telefone para teste
- Conhecimento básico de requisições HTTP

## Passo 1: Obter Seu Token de Ambiente

Seu token de ambiente é sua chave de API para autenticar requisições para a API Talkover.

### Como Obter Seu Token:

1. **Faça login no seu painel Talkover** em [app.talkover.ai](https://app.talkover.ai)
2. **Navegue para Conta** → **Ambientes**
3. **Copie seu Token de Ambiente** da lista (começa com `talq_`)

<Warning>
  Mantenha seu token de ambiente seguro e nunca o compartilhe publicamente. Trate-o como uma senha.
</Warning>

## Passo 2: Criar Seu Primeiro Agente de Voz

Agentes de voz são as personalidades de IA que gerenciam suas chamadas. Você precisa criar um antes de fazer chamadas.

### Como Criar um Agente de Voz:

1. **Vá para Agentes de Voz** no seu painel
2. **Clique em "Criar Novo Agente"**
3. **Configure seu agente:**
   - **Nome**: Dê um nome descritivo ao seu agente (ex: "Agente de Atendimento")
   - **Voz**: Escolha uma personalidade de voz
   - **Idioma**: Selecione o idioma para conversas
   - **Fluxo de Chamada**: Projete como seu agente deve gerenciar conversas
4. **Salve seu agente**
5. **Copie o ID do Agente** (você precisará disso para chamadas da API)

<Tip>
  Comece com um fluxo de chamada simples para teste. Você pode torná-lo mais complexo depois.
</Tip>

## Passo 3: Fazer Sua Primeira Chamada

Agora você está pronto para fazer sua primeira chamada usando a API Talkover!

### Endpoint da API

```
POST /api/v1/agents/{agent_id}/call
```

### Exemplo de Requisição

```bash
curl -X POST "https://app.talkover.ai/api/v1/agents/agent_123456/call" \
  -H "Authorization: Bearer talq_seu_token_de_ambiente_aqui" \
  -H "Content-Type: application/json" \
  -d '{
    "to": "+1234567890"
  }'
```

### Exemplo de Resposta

```json
{
  "call_id": "call_789012",
  "agent_id": "agent_123456",
  "to": "+1234567890",
  "status": "initiated",
  "created_at": "2024-01-15T10:30:00Z",
  "estimated_duration": 120
}
```

## Passo 4: Monitorar Sua Chamada

Após iniciar uma chamada, você pode:

1. **Verificar o status da chamada** no seu painel
2. **Ouvir gravações da chamada** (se habilitado)
3. **Ver transcrições da chamada** (se habilitado)
4. **Monitorar análises da chamada**
