# Acionar Importação SFTP

Aciona manualmente uma importação SFTP de novos arquivos de lista de contatos para uma campanha. A plataforma varre `import_path`, baixa novos arquivos e enfileira contatos.

## Endpoint

```
POST /api/v1/campaigns/{campaign_id}/sftp/trigger-import
```

## Parâmetros de Caminho

<ParamField path="campaign_id" type="string" required>UUID da campanha.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X POST "https://app.talkover.ai/api/v1/campaigns/campaign-uuid/sftp/trigger-import" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "SFTP import queued",
  "data": {
    "job_id": "job-uuid-1",
    "queued_at": "2024-01-15T17:00:00Z"
  }
}
```

</ResponseExample>

## Respostas de Erro

### 409 — Já em Execução

<ResponseExample>

```json
{
  "success": false,
  "message": "An SFTP import is already in progress",
  "code": "SFTP_IMPORT_IN_PROGRESS"
}
```

</ResponseExample>

## Observações

- O progresso e os resultados da importação aparecem em [Listar Logs SFTP](/api-reference/endpoints/list-campaign-sftp-logs).
