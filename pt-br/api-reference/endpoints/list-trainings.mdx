# Listar Treinamentos

Recuperar uma lista de dados de treinamento para um agente específico com suporte à paginação.

## Endpoint

```
GET /api/v1/agents/{agent_id}/trainings
```

## Parâmetros de Caminho

<ParamField path="agent" type="string" required>
  O identificador único do agente. Você pode encontrá-lo na lista de agentes.
</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Parâmetros de Consulta

<ParamField query="per_page" type="integer" required={false}>
  Número de itens por página (padrão: 20, máximo: 100)
</ParamField>

<ParamField query="page" type="integer" required={false}>
  Número da página (padrão: 1)
</ParamField>

## Exemplos de Requisição

<RequestExample>

```bash
# Requisição 1: Obter todos os treinamentos de um agente
curl -X GET "https://talkover.app/api/v1/agents/agent-uuid-1/trainings" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

```javascript
// Requisição 1: Obter todos os treinamentos de um agente
const response = await fetch('https://talkover.app/api/v1/agents/agent-uuid-1/trainings', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer talq_your_environment_token_here'
  }
});

const result = await response.json();
console.log(result);
```

</RequestExample>

<RequestExample>

```bash
# Requisição 2: Obter treinamentos com paginação
curl -X GET "https://talkover.app/api/v1/agents/agent-uuid-1/trainings?per_page=50&page=1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

```javascript
// Requisição 2: Obter treinamentos com paginação
const response = await fetch('https://talkover.app/api/v1/agents/agent-uuid-1/trainings?per_page=50&page=1', {
  method: 'GET',
  headers: {
    'Authorization': 'Bearer talq_your_environment_token_here'
  }
});

const result = await response.json();
console.log(result);
```

</RequestExample>

## Códigos de Erro

| Código | Descrição | Status HTTP |
|--------|-----------|-------------|
| `AGENT_NOT_FOUND` | Agente especificado não existe | 404 |
| `INVALID_TOKEN` | Token de autenticação inválido ou ausente | 401 |
| `SERVER_ERROR` | Erro interno do servidor ocorreu | 500 |

## Notas Importantes

<Info>
  **Suporte à paginação.** Use parâmetros `page` e `per_page` para navegar pelos resultados.
</Info>

<Info>
  **Dados de treinamento.** Os dados de treinamento ajudam a melhorar o desempenho do agente.
</Info>

<Info>
  **Perguntas e respostas.** Cada treinamento contém uma pergunta e sua resposta correspondente.
</Info>

## Melhores Práticas

1. **Use paginação** - Sempre use paginação para grandes conjuntos de dados de treinamento
2. **Mantenha dados atualizados** - Regularmente atualize os dados de treinamento para melhorar o desempenho
3. **Qualidade das perguntas** - Use perguntas claras e específicas para melhor treinamento
4. **Respostas detalhadas** - Forneça respostas completas e úteis
5. **Monitore desempenho** - Acompanhe como os dados de treinamento afetam o desempenho do agente

## Endpoints Relacionados

- **Criar/Atualizar Treinamento**: `POST /api/v1/agents/{agent_id}/trainings`
- **Excluir Treinamento**: `DELETE /api/v1/agents/{agent_id}/trainings/{training_id}`
- **Obter Agente**: `GET /api/v1/agents/{agent_id}`
- **Listar Agentes**: `GET /api/v1/agents`
