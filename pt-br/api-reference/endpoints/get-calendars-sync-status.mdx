# Obter Status de Sincronização dos Calendários

Obter o status atual de sincronização de todos os calendários do ambiente que possuem conexões com provedores.

## Endpoint

```
GET /api/v1/calendars/sync-status
```

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/calendars/sync-status" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": [
    {
      "calendar_id": "calendar-uuid-1",
      "provider": "calendar_provider_a",
      "last_sync_at": "2024-01-15T17:30:00Z",
      "is_syncing": false,
      "events_synced": 142
    }
  ]
}
```

</ResponseExample>
