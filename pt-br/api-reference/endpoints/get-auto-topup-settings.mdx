# Obter Configurações de Recarga Automática

Recupera a configuração atual de recarga automática (Auto Topup) do seu ambiente. A recarga automática repõe automaticamente o saldo da conta quando ele cai abaixo de um limite mínimo, ou em um dia agendado a cada mês.

## Endpoint

```
GET /api/v1/auto-topup/settings
```

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/auto-topup/settings" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "is_enabled": true,
    "threshold_amount": 50.00,
    "recharge_amount": 100.00,
    "scheduled_payment_enabled": false,
    "scheduled_amount": null,
    "day_of_month": null,
    "payment_method_id": "pm_uuid_1",
    "daily_limit": 500.00,
    "recharge_count_today": 0,
    "last_recharge_at": "2024-01-10T14:30:00Z",
    "currency": "USD"
  }
}
```

</ResponseExample>

### Campos da Resposta

<ResponseField name="data.is_enabled" type="boolean">Indica se a recarga automática está atualmente ativa.</ResponseField>
<ResponseField name="data.threshold_amount" type="number">Limite mínimo de saldo (na moeda da conta) abaixo do qual uma recarga é disparada.</ResponseField>
<ResponseField name="data.recharge_amount" type="number">Valor cobrado a cada vez que o limite mínimo é atingido.</ResponseField>
<ResponseField name="data.scheduled_payment_enabled" type="boolean">Indica se as recargas mensais agendadas estão ativas (independente do limite mínimo).</ResponseField>
<ResponseField name="data.scheduled_amount" type="number">Valor cobrado no dia agendado. Obrigatório quando `scheduled_payment_enabled` é true.</ResponseField>
<ResponseField name="data.day_of_month" type="integer">Dia do mês (1–28) para recargas agendadas.</ResponseField>
<ResponseField name="data.payment_method_id" type="string">ID do método de pagamento salvo usado para as cobranças.</ResponseField>
<ResponseField name="data.daily_limit" type="number">Valor total máximo que pode ser cobrado em um único dia em todos os eventos de recarga automática.</ResponseField>
<ResponseField name="data.recharge_count_today" type="integer">Número de recargas já executadas hoje.</ResponseField>
<ResponseField name="data.last_recharge_at" type="string">Timestamp ISO 8601 da recarga mais recente.</ResponseField>
