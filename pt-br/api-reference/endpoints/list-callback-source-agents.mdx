# Listar Agentes de Origem de Callback

Lista os agentes outbound que podem ser configurados como **agentes de origem** para um agente de callback. As chamadas outbound de um agente de origem se tornam candidatas para callback quando o contato retorna a ligação.

## Endpoint

```
GET /api/v1/callback-agents/source-agents
```

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/callback-agents/source-agents" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": [
    {
      "id": "agent-uuid-2",
      "name": "Sales Outbound",
      "direction": "outbound",
      "current_status": "published"
    },
    {
      "id": "agent-uuid-3",
      "name": "Survey Outbound",
      "direction": "outbound",
      "current_status": "published"
    }
  ]
}
```

</ResponseExample>

## Observações

- Apenas agentes `outbound` publicados são retornados.
- Use os IDs desta lista ao chamar [Definir Agente como Callback](/api-reference/endpoints/set-agent-as-callback) ou [Atualizar Origens de Callback](/api-reference/endpoints/update-callback-sources).
