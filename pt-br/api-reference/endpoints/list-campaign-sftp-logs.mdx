# Listar Logs SFTP da Campanha

Lista os jobs SFTP recentes de importação e exportação para uma campanha, incluindo status, contagem de arquivos e quaisquer erros.

## Endpoint

```
GET /api/v1/campaigns/{campaign_id}/sftp/logs
```

## Parâmetros de Caminho

<ParamField path="campaign_id" type="string" required>UUID da campanha.</ParamField>

## Parâmetros de Consulta

<ParamField query="per_page" type="integer">Itens por página. Faixa: `1`–`100`. Padrão: `25`.</ParamField>
<ParamField query="page" type="integer">Padrão: `1`.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/campaigns/campaign-uuid/sftp/logs" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

{
  "success": true,

</ResponseExample>

### Valores de Status

- `running` — job em andamento
- `succeeded` — concluído com sucesso
- `failed` — concluído com erros
- `cancelled` — cancelado manualmente
