# Cancelar Solicitação de Liberação

Cancela uma solicitação pendente de liberação de número de telefone antes que ela seja aprovada ou rejeitada. Uma vez cancelada, o número de telefone permanece em seu ambiente.

## Endpoint

```
DELETE /api/v1/phone-numbers/release-requests/{release_request_id}
```

## Parâmetros de Caminho

<ParamField path="release_request_id" type="string" required>
  UUID da solicitação de liberação a cancelar.
</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/phone-numbers/release-requests/request-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "Release request cancelled"
}
```

</ResponseExample>

## Respostas de Erro

### 404 — Não Encontrada

<ResponseExample>

```json
{
  "success": false,
  "message": "Release request not found"
}
```

</ResponseExample>

### 409 — Já Resolvida

<ResponseExample>

```json
{
  "success": false,
  "message": "Release request already resolved and cannot be cancelled",
  "code": "RELEASE_ALREADY_RESOLVED"
}
```

</ResponseExample>

## Observações

- Apenas solicitações com status `pending` podem ser canceladas.
- Cancelar uma solicitação não a exclui — seu status é atualizado para `cancelled` e ela permanece em [Listar Solicitações de Liberação](/api-reference/endpoints/list-release-requests).
