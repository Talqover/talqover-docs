# Listar Versões do Fluxo do Agente

Lista as versões históricas da configuração de fluxo de um agente. Cada salvamento gera uma entrada de versão que pode ser inspecionada.

## Endpoint

```
GET /api/v1/agents/{agent_id}/flow/versions
```

## Parâmetros de Caminho

<ParamField path="agent_id" type="string" required>UUID do agente.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://talkover.app/api/v1/agents/agent-uuid/flow/versions" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "versions": [
      {
        "id": "version-uuid-3",
        "version": "1.2.0",
        "created_at": "2024-01-15T10:00:00Z",
        "is_current": true
      },
      {
        "id": "version-uuid-2",
        "version": "1.1.0",
        "created_at": "2024-01-10T14:00:00Z",
        "is_current": false
      },
      {
        "id": "version-uuid-1",
        "version": "1.0.0",
        "created_at": "2024-01-05T09:00:00Z",
        "is_current": false

  }

```

</ResponseExample>

## Observações

- O versionamento é automático em cada atualização de fluxo.
- Restaurar uma versão anterior requer buscar seu `flow_config` e enviá-lo via [Atualizar Fluxo do Agente](/api-reference/endpoints/update-agent-flow).
