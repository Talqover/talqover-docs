# Remover Papel de Callback

Remove o papel de callback de um agente. O agente para de tratar callbacks mas, no restante, permanece ativo para o tráfego inbound normal.

## Endpoint

```
DELETE /api/v1/agents/{agent_id}/callback
```

## Parâmetros de Caminho

<ParamField path="agent_id" type="string" required>UUID do agente de callback.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/agents/agent-uuid-1/callback" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "Callback role removed"
}
```

</ResponseExample>

## Observações

- As demais configurações do agente (conhecimento, voz, números de telefone) são preservadas.
- Chamadas inbound de contatos previamente roteados via callback agora seguirão o roteamento padrão para o número discado.
