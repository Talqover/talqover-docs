# Obter Usuário Atual

Recupera o perfil do usuário autenticado. Este endpoint requer um **token de sessão de usuário** (API de Conta), não um token de ambiente.

<Info>
  Este endpoint faz parte da **API de Conta** e é autenticado com um token de sessão de usuário. Outros endpoints desta referência usam tokens de ambiente (`talq_*`).
</Info>

## Endpoint

```
GET /api/user
```

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token de sessão do usuário. Formato: `Bearer <user_session_token>`.
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/user" \
  -H "Authorization: Bearer your_user_session_token"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "id": "user-uuid-1",
  "name": "Alice Doe",
  "email": "alice@example.com",
  "current_environment_id": "env-uuid-1",
  "created_at": "2024-01-01T10:00:00Z"
}
```

</ResponseExample>

## Observações

- Use este endpoint em dashboards ou ferramentas que autenticam o usuário humano, não integrações servidor-a-servidor.
- Para acesso máquina-a-máquina com escopo de ambiente, use tokens de ambiente (`talq_*`) com o restante da API.
