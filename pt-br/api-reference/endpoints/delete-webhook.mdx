---
title: "Excluir Webhook"
api: "DELETE /api/v1/agents/{agent_id}/webhooks/{webhook_id}"
description: "Excluir permanentemente uma configuração de webhook"
---

## Visão Geral

Exclui permanentemente uma configuração de webhook. Esta ação não pode ser desfeita.

## Endpoint

<ParamField path="agent_id" type="string" required>
  UUID do Agente
</ParamField>

<ParamField path="webhook_id" type="string" required>
  UUID do Webhook
</ParamField>

## Notas Importantes

- Esta ação exclui permanentemente a configuração do webhook e não pode ser desfeita
- Operações de webhook limpam automaticamente o cache do agente para efeito imediato
- Considere desabilitar o webhook primeiro se quiser parar temporariamente as notificações
- Webhooks excluídos não receberão mais nenhum evento
