# Remover Membro do Calendário

Revogar o acesso de um usuário a um calendário.

## Endpoint

```
DELETE /api/v1/calendars/{calendar_id}/members/{member_id}
```

## Parâmetros de caminho

<ParamField path="calendar_id" type="string" required>UUID do calendário.</ParamField>
<ParamField path="member_id" type="string" required>UUID do membro.</ParamField>

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/calendars/calendar-uuid-1/members/member-uuid-2" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "Member removed"
}
```

</ResponseExample>

## Observações

- O `owner` do calendário não pode ser removido — transfira a propriedade primeiro via [Atualizar Membro do Calendário](/api-reference/endpoints/update-calendar-member).
