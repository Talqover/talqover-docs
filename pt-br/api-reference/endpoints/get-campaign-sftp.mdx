# Obter Configuração SFTP da Campanha

Recupera a configuração de integração SFTP para uma campanha. O SFTP de campanha permite importar listas de contatos de um servidor SFTP remoto e exportar resultados de chamadas de volta.

## Endpoint

```
GET /api/v1/campaigns/{campaign_id}/sftp
```

## Parâmetros de Caminho

<ParamField path="campaign_id" type="string" required>UUID da campanha.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/campaigns/campaign-uuid/sftp" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "id": "sftp-config-uuid-1",
    "campaign_id": "campaign-uuid",
    "host": "sftp.example.com",
    "port": 22,
    "username": "campaigns_user",
    "auth_type": "password",
    "import_path": "/inbox",
    "export_path": "/outbox",
    "import_schedule": "daily",
    "is_enabled": false,
    "last_test_at": "2024-01-10T14:30:00Z",
    "last_test_success": true,
    "created_at": "2024-01-01T10:00:00Z"
  }
}
```

</ResponseExample>

## Observações

- O `password` ou `private_key` configurado nunca é retornado.
- Uma nova configuração SFTP começa desabilitada — execute [Testar Conexão SFTP](/api-reference/endpoints/test-campaign-sftp-connection) com sucesso antes de habilitá-la.
