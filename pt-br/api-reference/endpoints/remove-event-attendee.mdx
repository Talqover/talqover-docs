# Remover Participante do Evento

Remover um participante de um evento de calendário.

## Endpoint

```
DELETE /api/v1/calendars/{calendar_id}/events/{event_id}/attendees/{attendee_id}
```

## Parâmetros de caminho

<ParamField path="calendar_id" type="string" required>UUID do calendário.</ParamField>
<ParamField path="event_id" type="string" required>UUID do evento.</ParamField>
<ParamField path="attendee_id" type="string" required>UUID do participante.</ParamField>

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/calendars/calendar-uuid-1/events/event-uuid-1/attendees/att-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "Attendee removed"
}
```

</ResponseExample>
