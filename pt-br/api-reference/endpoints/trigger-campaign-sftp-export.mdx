# Acionar Exportação SFTP

Exporta arquivos de resultado de campanha selecionados para o `export_path` configurado no servidor SFTP.

## Endpoint

```
POST /api/v1/campaigns/{campaign_id}/sftp/trigger-export
```

## Parâmetros de caminho

<ParamField path="campaign_id" type="string" required>UUID da campanha.</ParamField>

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

<ParamField header="Content-Type" type="string" required>
  Deve ser definido como `application/json`
</ParamField>

## Corpo da requisição

<ParamField body="file_data_ids" type="array" required>
  Array de UUIDs de arquivos a exportar. Use [Obter Campanha](/api-reference/endpoints/get-campaign) para recuperar os `campaign_files` disponíveis.
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X POST "https://app.talkover.ai/api/v1/campaigns/campaign-uuid/sftp/trigger-export" \
  -H "Authorization: Bearer talq_your_environment_token_here" \
  -H "Content-Type: application/json" \
  -d '{
    "file_data_ids": ["file-uuid-1", "file-uuid-2"]
  }'
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "SFTP export queued",
  "data": {
    "job_id": "job-uuid-2",
    "files_count": 2
  }
}
```

</ResponseExample>

## Respostas de erro

### 409 — Já em Execução

<ResponseExample>

```json
{
  "success": false,
  "message": "An SFTP export is already in progress",
  "code": "SFTP_EXPORT_IN_PROGRESS"
}
```

</ResponseExample>
