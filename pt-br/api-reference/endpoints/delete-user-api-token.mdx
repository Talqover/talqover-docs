# Excluir Token de API do Usuário

Revoga um Personal Access Token. O token é invalidado imediatamente e quaisquer requisições subsequentes que o utilizem retornam `401`.

<Info>
  Este endpoint faz parte da **API de Conta** e é autenticado com um token de sessão de usuário.
</Info>

## Endpoint

```
DELETE /api/v1/user/api-tokens/{token_id}
```

## Parâmetros de Caminho

<ParamField path="token_id" type="string" required>
  ID do token a ser revogado. Use [Listar Tokens de API do Usuário](/api-reference/endpoints/list-user-api-tokens) para encontrar o ID.
</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token de sessão do usuário. Formato: `Bearer <user_session_token>`.
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/user/api-tokens/token-uuid-1" \
  -H "Authorization: Bearer your_user_session_token"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "message": "Token revoked"
}
```

</ResponseExample>

## Observações

- A revogação é imediata e irreversível.
- Requisições já em curso usando o token revogado serão concluídas; novas requisições falharão com `401`.
