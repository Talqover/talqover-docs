# Excluir DID do SIP Trunk

Remove um DID de um SIP Trunk. O DID também é removido do conjunto de números de telefone do seu ambiente. Desassocie-o de quaisquer agentes primeiro.

## Endpoint

```
DELETE /api/v1/sip-trunks/{trunk_id}/dids/{phone_number_id}
```

## Parâmetros de Caminho

<ParamField path="trunk_id" type="string" required>UUID do SIP Trunk.</ParamField>
<ParamField path="phone_number_id" type="string" required>UUID do DID/número de telefone.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X DELETE "https://app.talkover.ai/api/v1/sip-trunks/trunk-uuid-1/dids/phone-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "message": "DID removed"
}
```

</ResponseExample>

## Respostas de Erro

### 409 — DID Em Uso

<ResponseExample>

```json
{
  "success": false,
  "message": "DID is still assigned to active agents",
  "code": "DID_IN_USE"
}
```

</ResponseExample>
