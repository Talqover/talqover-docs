# Health Check

Verificação de liveness leve. Retorna `200 OK` quando a API está acessível. Use para monitores de uptime ou health checks de balanceadores de carga.

## Endpoint

```
GET /api/v1/health
```

## Autenticação

**Nenhuma.** Este endpoint não requer autenticação.

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/health"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "status": "ok"
}
```

</ResponseExample>

## Observações

- Esta é uma verificação de liveness apenas — ela não exercita dependências downstream.
- Seguro para polling frequente; o endpoint é isento de rate limit.
