# Atualizar Membro do Calendário

Alterar a função de um membro do calendário.

## Endpoint

```
PUT /api/v1/calendars/{calendar_id}/members/{member_id}
```

## Parâmetros de Caminho

<ParamField path="calendar_id" type="string" required>UUID do calendário.</ParamField>
<ParamField path="member_id" type="string" required>UUID do membro.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

<ParamField header="Content-Type" type="string" required>Deve ser `application/json`.</ParamField>

## Corpo da Requisição

<ParamField body="role" type="string" required>Nova função. Opções: `owner`, `editor`, `viewer`.</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X PUT "https://app.talkover.ai/api/v1/calendars/calendar-uuid-1/members/member-uuid-2" \
  -H "Authorization: Bearer talq_your_environment_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "role": "viewer" }'
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "id": "member-uuid-2",
    "role": "viewer"
  }
}
```

</ResponseExample>
