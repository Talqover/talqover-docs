# Disparar Sincronização de Calendários

Dispara uma sincronização de eventos das integrações com provedores de calendário conectados. Executa de forma assíncrona — consulte [Obter Status de Sincronização](/api-reference/endpoints/get-calendars-sync-status) para acompanhar o progresso.

## Endpoint

```
POST /api/v1/calendars/sync
```

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl -X POST "https://app.talkover.ai/api/v1/calendars/sync" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "syncing": ["calendar-uuid-1", "calendar-uuid-2"],
    "up_to_date": ["calendar-uuid-3"]
  }
}
```

</ResponseExample>
