# Listar Calendários

Listar calendários no ambiente. Calendários armazenam eventos, membros e conexões com provedores — normalmente usados por agentes que agendam compromissos.

## Endpoint

```
GET /api/v1/calendars
```

## Parâmetros de Consulta

<ParamField query="per_page" type="integer">Itens por página. Intervalo: `1`–`100`. Padrão: `25`.</ParamField>

## Cabeçalhos da Requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/calendars" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Resposta

<ResponseExample>

{
  "success": true,
