# Listar SIP Trunks

Lista os SIP Trunks personalizados configurados no seu ambiente. SIP Trunks personalizados permitem que clientes Enterprise roteiem chamadas pela própria operadora em vez do transporte padrão gerenciado pela plataforma.

<Info>
  Este recurso está disponível apenas em planos onde `allow_custom_sip_trunks` está habilitado. Entre em contato com o gerente da sua conta para habilitá-lo.
</Info>

## Endpoint

```
GET /api/v1/sip-trunks
```

## Cabeçalhos da requisição

<ParamField header="Authorization" type="string" required>
  Token Bearer para autenticação. Formato: `Bearer talq_your_environment_token_here`
</ParamField>

## Exemplos

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/sip-trunks" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>
