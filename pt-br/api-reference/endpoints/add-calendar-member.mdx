# Add Calendar Member

Grant a user access to a calendar with a specific role.

## Endpoint

```
POST /api/v1/calendars/{calendar_id}/members
```

## Path Parameters

<ParamField path="calendar_id" type="string" required>Calendar UUID.</ParamField>

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

<ParamField header="Content-Type" type="string" required>Must be `application/json`.</ParamField>

## Request Body

<ParamField body="user_id" type="string" required>UUID of the user to add.</ParamField>
<ParamField body="role" type="string" required>Member role. Options: `owner`, `editor`, `viewer`.</ParamField>

## Examples

<RequestExample>

```bash
curl -X POST "https://talkover.app/api/v1/calendars/calendar-uuid-1/members" \
  -H "Authorization: Bearer talq_your_environment_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "user_id": "user-uuid-2", "role": "editor" }'
```

</RequestExample>

## Response

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "id": "member-uuid-2",
    "user_id": "user-uuid-2",
    "role": "editor"
  }
}
```

</ResponseExample>
