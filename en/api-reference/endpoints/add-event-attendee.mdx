# Add Event Attendee

Add an attendee to a calendar event.

## Endpoint

```
POST /api/v1/calendars/{calendar_id}/events/{event_id}/attendees
```

## Path parameters

<ParamField path="calendar_id" type="string" required>Calendar UUID.</ParamField>
<ParamField path="event_id" type="string" required>Event UUID.</ParamField>

## Request headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

<ParamField header="Content-Type" type="string" required>Must be `application/json`.</ParamField>

## Request body

<ParamField body="name" type="string" required>Attendee display name.</ParamField>
<ParamField body="email" type="string" required>Attendee email.</ParamField>

## Examples

<RequestExample>

```bash
curl -X POST "https://app.talkover.ai/api/v1/calendars/calendar-uuid-1/events/event-uuid-1/attendees" \
  -H "Authorization: Bearer talq_your_environment_token_here" \
  -H "Content-Type: application/json" \
  -d '{ "name": "Jane Doe", "email": "jane@acme.com" }'
```

</RequestExample>

## Response

<ResponseExample>

```json
{
  "success": true,
  "data": {
    "id": "att-uuid-1",
    "name": "Jane Doe",
    "email": "jane@acme.com"
  }
}
```

</ResponseExample>
