# Get Campaign Webhook

Retrieve a campaign webhook configuration.

## Endpoint

```
GET /api/v1/campaigns/{campaign_id}/webhooks/{webhook_id}
```

## Path Parameters

<ParamField path="campaign_id" type="string" required>Campaign UUID.</ParamField>
<ParamField path="webhook_id" type="string" required>Webhook UUID.</ParamField>

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

## Examples

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/campaigns/campaign-uuid/webhooks/webhook-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>
