# Get Auto Topup History

List recent auto topup recharge events for your environment, including amount, status, and the trigger (threshold or scheduled).

## Endpoint

```
GET /api/v1/auto-topup/history
```

## Query Parameters

<ParamField query="per_page" type="integer">Items per page. Range: `1`–`100`. Default: `25`.</ParamField>
<ParamField query="page" type="integer">Page number. Default: `1`.</ParamField>

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

## Examples

<RequestExample>

```bash
curl "https://talkover.app/api/v1/auto-topup/history?per_page=10" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

### Trigger Values

- `threshold` — fired because balance dropped below `threshold_amount`
- `scheduled` — fired on a configured day of month
- `test` — manual test recharge via [Test Auto Topup](/api-reference/endpoints/test-auto-topup)

### Status Values

- `pending` — payment in flight
- `succeeded` — payment captured, balance credited
- `failed` — payment declined or errored
