# Get Calendar Event

Retrieve a single event from a calendar, including attendees.

## Endpoint

```
GET /api/v1/calendars/{calendar_id}/events/{event_id}
```

## Path parameters

<ParamField path="calendar_id" type="string" required>Calendar UUID.</ParamField>
<ParamField path="event_id" type="string" required>Event UUID.</ParamField>

## Request headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

## Examples

<RequestExample>

```bash
curl "https://app.talkover.ai/api/v1/calendars/calendar-uuid-1/events/event-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>
