# Delete Calendar

Permanently delete a calendar, along with its events, members, availability rules, and provider connections.

<Warning>
  This is irreversible.
</Warning>

## Endpoint

```
DELETE /api/v1/calendars/{calendar_id}
```

## Path Parameters

<ParamField path="calendar_id" type="string" required>Calendar UUID.</ParamField>

## Request Headers

<ParamField header="Authorization" type="string" required>
  Bearer token. Format: `Bearer talq_your_environment_token_here`
</ParamField>

## Examples

<RequestExample>

```bash
curl -X DELETE "https://talkover.app/api/v1/calendars/calendar-uuid-1" \
  -H "Authorization: Bearer talq_your_environment_token_here"
```

</RequestExample>

## Response

<ResponseExample>

```json
{
  "success": true,
  "message": "Calendar deleted"
}
```

</ResponseExample>
